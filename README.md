# node-express-board

Board Back-End Logic Design based Node.js &amp; Express.js

## 요구사항

> DB 없이 배열에 담는다고 생각하고 시작

1. Post

- Post를 등록할 수 있다.
- Post의 목록을 조회할 수 있다.
- Post를 수정할 수 있다.
- Post의 작성 양식에 맞지 않으면 등록 / 수정 할 수 없다.
  - 작성자가 있어야한다.
  - 작성일 / 수정일이 있어야한다.
  - 작성내용이 있어야한다.
  - 특정 Board에 속해야한다.
- Post를 제거할 수 있다.

2. Board

- Board를 등록할 수 있다.
- Board의 목록을 조회할 수 있다.
